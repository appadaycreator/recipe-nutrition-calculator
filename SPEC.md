# レシピ栄養計算ツール【無料】簡単・無料計算 - 技術仕様書

## 概要

**サービス名**: Recipe Nutrition Calculator
**バージョン**: 2.1.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/recipe-nutrition-calculator/

食材と量を入力してカロリー・タンパク質・脂質・炭水化物を自動計算。献立の栄養バランスチェックに。登録不要・完全無料でご利用いただけます。

## 1日推奨摂取量の基準

成人平均を基準値として使用:
- カロリー: 2,000 kcal
- タンパク質: 60 g
- 脂質: 55 g
- 炭水化物: 300 g

## データ管理

- **ストレージ**: 計算はブラウザ内のみ（外部API通信なし）
- **永続化**: localStorage使用
  - `recipe_nutrition_data`: 現在の入力値（自動保存・復元）
  - `recipe_nutrition_history`: 計算履歴（最新10件）

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- **Chart.js** 4.4.1（PFC比率グラフ）
- **localStorage** API（入力値・履歴保存）
- PWA対応（manifest.json）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ドロップダウンから食材を選ぶ（35種類以上、カテゴリ別）
2. 食材の量をグラム単位で入力する（例: 鶏胸肉 150g）
3. 「＋ 食材を追加」で複数の食材を追加する（最大10食材）
4. 「栄養素を計算する」ボタンをタップして結果を確認する
5. カロリーとPFCの合計と1日の推奨量に対する割合を確認する

## よくある質問（FAQ）

**Q: レシピ栄養計算ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何種類の食材が登録されていますか？**

肉類・魚介類・野菜類・穀物豆類・乳製品・油脂など35種類以上の食材が登録されています。

**Q: 1日の推奨量はどのような基準ですか？**

成人平均を基準にカロリー2,000kcal、タンパク質60g、脂質55g、炭水化物300gを目安にしています。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

計算後はSNSシェアボタンに計算結果が自動反映されます。X（旧Twitter）やLINEでシェアできます。
